🚀 راهنمای شروع سریع

  1. با استفاده از SSH به سرور متصل شوید
  2. رمز عبور root را تغییر دهید
  3. سیستم را بروزرسانی کنید:
    apt update && apt upgrade -y # Ubuntu/Debian
    yum update -y # CentOS
  4. Firewall را تنظیم کنید
  5. نرم‌افزارهای مورد نیاز را نصب کنید
مدیریت سرور در پنل کاربری
